What does this project do?

This system applies Natural Language Processing and Machine Learning to analyze speech patterns and predict the most likely political speaker. The model focuses on how something is said, not what position is taken.

How it works

1. Input Speech

User provides raw political speech text.

2. Feature Extraction

TF-IDF captures vocabulary, phrasing, and repetition patterns.

3. Classification

A trained Logistic Regression model outputs speaker probabilities.
